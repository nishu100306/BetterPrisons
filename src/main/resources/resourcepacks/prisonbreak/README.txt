PB Pack (1.21.11)

Changes these block textures to a soft neutral tan-white theme:
- redstone_ore / redstone_block
- iron_ore / iron_block
- lapis_ore / lapis_block
- diamond_ore / diamond_block
- emerald_ore / emerald_block
- coal_ore / coal_block
- gold_ore / gold_block
- stone

Ore blocks and Ore have slightly different textures so you know what to prioritize if you choose to do so.

Install:
1. Put the .zip in your .minecraft/resourcepacks folder
2. Open Minecraft
3. Options -> Resource Packs
4. Enable
